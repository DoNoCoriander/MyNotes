# Node.js

### 一.命令

```react
npm install -g cnpm --registry=https://registry.npm.taobao.org
// 使用淘宝镜像的命令
npm list antd
// 查看antd的版本
npm list -g
// 查看所有全局安装的模块
npm update antd
// 更新antd模块
npm init
// 生成自己的package.json文件
npm adduser
// 在npm的资源库中注册用户
npm publish
// 发布自己的模块
npm cache clean
// 安装node_modules报错npm resource busy or locked.... 可以使用此命令清除之前安装的node_modules
```
